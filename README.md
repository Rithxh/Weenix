# Weenix
A Unix-like operating system kernel developed as part of the graduate-level **Operating Systems (CS402)** course. The project explores core OS design concepts through a modular, monolithic kernel written in C.

**Features**
- Kernel threads, processes, and synchronization  
- Virtual memory with demand paging and copy-on-write  
- VFS and System V file system (S5FS)  
- TTY, disk, and memory device support

This repository is a placeholder to describe the project. The full source code is not publicly available to comply with course academic integrity policies.
